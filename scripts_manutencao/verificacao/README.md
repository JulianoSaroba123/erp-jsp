# Scripts de Verificação

Scripts para verificar dados, estrutura do banco e status do sistema.

## Quando usar:
- Verificar integridade dos dados
- Listar registros do banco
- Comparar estruturas entre ambientes
- Validar configurações

## Arquivos principais:
- `verificar_*.py` - Verificações específicas
- `ver_*.py` - Visualização de estruturas
- `listar_*.py` - Listagem de dados
- `find_*.py` - Busca de padrões

**NOTA:** Estes scripts apenas leem dados, não fazem alterações.
