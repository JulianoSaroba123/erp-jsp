# Scripts de Correção

Scripts para corrigir dados, recalcular valores e resetar configurações.

## Quando usar:
- Recalcular custos e valores após alterações
- Resetar senhas de admin
- Normalizar dados
- Extrair informações de templates
- Gerar ícones e assets

## Arquivos principais:
- `recalcular_*.py` - Recálculo de valores
- `resetar_*.py` - Reset de configurações
- `normalize_*.py` - Normalização de dados
- `extrair_*.py` - Extração de informações
- `gerar_*.py` - Geração de assets

**AVISO:** Alguns scripts modificam dados. Execute com cuidado e faça backup antes.
