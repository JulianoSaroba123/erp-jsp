# Scripts de Dados Exemplo

Scripts para popular o banco de dados com dados de exemplo/teste.

## Quando usar:
- Configuração inicial do sistema
- Ambiente de desenvolvimento/teste
- Demonstrações
- Testes de funcionalidades

## Arquivos principais:
- `criar_*.py` - Criação de registros exemplo
- Cria: colaboradores, equipamentos, serviços, contas bancárias, etc.

**AVISO:** Use apenas em ambiente de desenvolvimento! Não execute em produção sem revisar os dados.
