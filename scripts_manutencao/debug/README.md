# Scripts de Debug

Scripts utilizados para debugar e diagnosticar problemas no sistema.

## Quando usar:
- Problemas de login ou autenticação
- Erros em módulos específicos (cliente, proposta, projeto)
- Diagnóstico de problemas no Render
- Verificação de queries SQL

## Arquivos principais:
- `debug_*.py` - Scripts de debug específicos por módulo
- `diagnosticar_*.py` - Scripts de diagnóstico completo

**NOTA:** Estes scripts são apenas para desenvolvimento/troubleshooting e não afetam o funcionamento do sistema em produção.
