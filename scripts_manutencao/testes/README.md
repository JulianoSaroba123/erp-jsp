# Scripts de Testes

Scripts para testar funcionalidades específicas do sistema.

## Quando usar:
- Testar integração entre módulos
- Validar funcionalidades após alterações
- Testes de geração de documentos
- Testes de APIs externas

## Arquivos principais:
- `test_*.py` - Testes de funcionalidades
- `testar_*.py` - Testes específicos de features

**NOTA:** Scripts de desenvolvimento. Podem ser executados sem riscos ao banco de produção.
