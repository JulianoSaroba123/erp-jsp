# Scripts SQL e Shell

Scripts SQL e shell scripts para manutenção do banco de dados.

## Quando usar:
- Executar comandos SQL diretos no banco
- Automação de tarefas no servidor
- Correções estruturais no banco

## Arquivos:
- `*.sql` - Scripts SQL
- `*.sh` - Shell scripts (Linux/Render)
- `*.bat` - Batch scripts (Windows)
- `*.ps1` - PowerShell scripts

**AVISO:** Scripts SQL modificam o banco diretamente. Use com muito cuidado!
