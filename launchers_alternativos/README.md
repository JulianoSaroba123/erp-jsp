# Launchers Alternativos

Versões alternativas de launchers para iniciar o sistema.

## Launcher Principal (em uso):
**🚀 INICIAR ERP JSP.bat** (na pasta raiz)

## Launchers Alternativos (aqui):
- `launcher*.py` - Versões Python dos launchers
- `jsp_launcher*.py` - Launchers JSP específicos
- `*.bat` - Batch launchers alternativos

## Quando usar:
Estes launchers são versões alternativas/antigas. Use apenas se o launcher principal apresentar problemas.

**NOTA:** O launcher recomendado é o `🚀 INICIAR ERP JSP.bat` na raiz do projeto.
